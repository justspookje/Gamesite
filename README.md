# 🕹️ PIXEL ARCADE

A complete browser-based game site built for **GitHub Pages** — no install, no build step, pure HTML/CSS/JS.

## 🎮 Games Included

| Game | Description |
|------|-------------|
| ✊ Rock Paper Scissors | VS AI — Best of 3, 5, 10, or Endless |
| 🪢 Hangman | 5 categories, 100+ words, score tracking |
| 🧠 Trivia Blast | 6 categories, 60+ questions, timed rounds |
| 🏓 Pong | Classic arcade Pong, 1P vs AI or 2P local |
| ♟️ Chess | Full chess with legal move validation |
| ⛳ Mini Golf | 6 unique holes, physics, power meter |
| 🐍 Snake | Classic Snake with levels and high score |
| 🃏 Memory Match | Card matching, 4×4, 5×4, or 6×6 grids |
| ❎ Tic Tac Toe | VS AI (minimax) or 2 player |
| 🧱 Breakout | Ball + paddle + bricks, multi-level |
| 🔤 Word Guess | 5-letter word puzzle, Wordle-style |

## 🚀 Deploy to GitHub Pages

1. **Fork or clone** this repository
2. Go to your repo **Settings → Pages**
3. Set source to **main branch / root folder**
4. Visit `https://yourusername.github.io/your-repo-name`

That's it! All games run entirely in the browser.

## 🎨 Design

- **Aesthetic**: Neon retro arcade / CRT scanlines
- **Font**: Press Start 2P (Google Fonts)
- **Colors**: Deep space background with neon cyan, pink, yellow accents
- **Mobile**: Responsive with touch controls on all games

## 🛠️ Tech Stack

- Pure **HTML5 / CSS3 / Vanilla JavaScript**
- Canvas API for Pong, Snake, Golf, Breakout
- No frameworks, no dependencies, no build tools
